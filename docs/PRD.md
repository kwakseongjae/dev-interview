# DevInterview - 개발자 기술면접 준비 서비스 PRD

## 1. 개요

### 1.1 프로젝트 목적

개발자들이 기술면접을 효과적으로 준비할 수 있도록 AI 기반 맞춤형 질문 생성 및 모의 면접 환경을 제공하는 웹 서비스

### 1.2 핵심 가치

- **맞춤형 질문 생성**: 경력, 포지션, 기술 스택에 맞는 면접 질문 자동 생성
- **실전 연습 환경**: 타이머, 힌트 기능을 통한 실제 면접과 유사한 환경 제공
- **학습 아카이브**: 과거 연습 기록을 저장하고 복습할 수 있는 시스템

### 1.3 기술 스택

- **Frontend**: Next.js 14 (App Router), TypeScript, Tailwind CSS
- **AI Integration**: Claude API (Anthropic)
- **Database**: Supabase (향후 연동 예정)
- **Storage**: LocalStorage (초기 버전)

## 2. 사용자 플로우

```
[랜딩 페이지] → [검색/질문 생성] → [질문 확인] → [모의 면접] → [제출 완료] → [아카이브]
```

### 2.1 상세 플로우

1. 랜딩 페이지 진입
2. 하단 입력창에 면접 조건 입력 (예: "프론트엔드 3년차 개발자")
3. AI가 검색 UI와 함께 질문 리스트 5개 생성
4. "이 질문으로 면접 준비하기" 버튼 클릭
5. 모의 면접 페이지로 이동
6. 각 질문에 3분씩 답변 작성 (힌트 기능 제공)
7. 5개 질문 완료 후 제출
8. 결과 아카이브에 저장

## 4. 기능 명세

### 4.1 질문 생성 기능

| 항목      | 설명                                       |
| --------- | ------------------------------------------ |
| 입력      | 포지션, 경력, 기술스택 등 자유 형식 텍스트 |
| 처리      | Claude API를 통한 맞춤형 질문 생성         |
| 출력      | 5개의 면접 질문 리스트 + 각 질문별 힌트    |
| 데모 모드 | 아무 값이나 입력해도 샘플 질문 반환        |

### 4.2 모의 면접 기능

| 항목     | 설명                                 |
| -------- | ------------------------------------ |
| 타이머   | 질문당 3분 (180초) 카운트다운        |
| 힌트     | 각 질문별 키워드 기반 힌트 제공      |
| 임시저장 | 다음 버튼 클릭 시 현재 답변 임시저장 |
| 제출     | 5개 질문 완료 후 최종 제출           |

### 4.3 데이터 저장 기능

| 항목      | 설명                                |
| --------- | ----------------------------------- |
| 저장 위치 | LocalStorage (향후 Supabase 연동)   |
| 저장 항목 | 질문, 답변, 소요시간, 날짜, 찜 여부 |
| 아카이브  | 과거 면접 기록 조회 및 재시험       |
| 찜하기    | 개별 질문 북마크 기능               |

## 5. 데이터 구조

### 5.1 Interview Session

```typescript
interface InterviewSession {
  id: string;
  createdAt: string;
  query: string; // 사용자 검색 쿼리
  questions: Question[];
  totalTime: number; // 총 소요 시간 (초)
  isCompleted: boolean;
}

interface Question {
  id: string;
  content: string; // 질문 내용
  hint: string; // 힌트 키워드
  answer: string; // 사용자 답변
  timeSpent: number; // 해당 질문 소요 시간
  isAnswered: boolean;
  isFavorite: boolean; // 찜 여부
}
```
